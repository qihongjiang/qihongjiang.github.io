Project

The application is a Java-based logo maker that allows users to add shapes, texts, and images. The GUI was created using JavaFX and the app itself was created using a app framework provided.

The application provides user interface of changing color gradients and saves data as one JSON object. Nearly every part of the app is created using a framework which regulates the colors and size of each object, therefore allowing the user of the framework an easy way to change many portions of the app.
