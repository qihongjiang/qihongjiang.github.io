Project

The application is a Java-based logo maker that allows users to add shapes, texts, and images. The GUI was created using JavaFX and the app itself was created using a app framework provided.

The application provides user interface of changing color gradients and saves data as one JSON object. Nearly every part of the app is created using a framework which regulates the colors and size of each object, therefore allowing the user of the framework an easy way to change many portions of the app.

Since the register function has not been implemented, the users are provided with a account of customer, customer representative, or manager and a password that is already set up in the database.

In the trading system, the customer can place order, track order history, and search for stocks. A manager can manage employees that serve the customers, functions including add/view/edit/delete. A manager can also view income reports, set stock price, and view stocks/bestsellers. A customer representative is an employee that manages customers and can also view the customer mailing list.

This is a project that I have completed while I was TAing for CSE-219, an advanced Java programming courses that use JavaFX. Since it is a side project, the icons for the user interface are more of a personal preference instead of professional icons.

The application has five tabs and the last four tabs correspond to four different pages of the website that it generates. For the first tab, the user can select the attributes for the page styling, and the contents are determined by the users' selections on the other four tabs.
